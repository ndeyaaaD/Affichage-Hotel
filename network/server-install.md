# Guide d'installation Serveur VM (Ubuntu)

1. **Système :** Installation d'Ubuntu Server 22.04 LTS sur VirtualBox.
2. **Réseau :** Configuration d'une IP statique `192.168.1.45`.
3. **Services :** - Mise à jour : `sudo apt update`
   - Installation Nginx : `sudo apt install nginx`
4. **Déploiement :** Transfert des fichiers du dashboard dans `/var/www/html`.
5. **Test :** Vérification de l'accès depuis l'écran 1 via le navigateur.
