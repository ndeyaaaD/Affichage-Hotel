
# Procédure de dépannage – Méthode Couche par Couche

| Étape | Vérification | Commande |
| :--- | :--- | :--- |
| 1 | Physique | Vérifier les voyants verts dans Packet Tracer |
| 2 | IP | `ipconfig` sur les écrans / `ip a` sur le serveur |
| 3 | Passerelle | `ping 192.168.1.1` |
| 4 | VLAN | `show vlan brief` sur le Switch |
| 5 | ACL | `show access-lists` sur le Routeur |
| 6 | Service | `curl http://192.168.1.45` |
