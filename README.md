# Projet Affichage Digital - Hôtel Le Ndiambour
Cette architecture réseau segmente le parc informatique en 3 VLANs (Admin, Display, Server) pour garantir la sécurité et la performance.
L'affichage est géré par un serveur Ubuntu/Nginx diffusant vers 10 écrans connectés.
La topologie est simulée sous Cisco Packet Tracer avec un routage inter-VLAN.
